# Status Tracker

An API to grab interesting status information

## Why

I'm building a personal website and thought it'd be cool if I could have live-ish information in the "about me" section, so I'm building this out.

Also coincidentally saw [this raspapi](https://raspapi.hackclub.com) hackclub initive so I decided to build a fully featured API

## Endpoints

##### Database Queries

GET /api/user/{:id} 

##### Unsaved Queries

GET /api/{endpoint}/{endpoint id}/{supported item}/{optional parameters}
